<hr><h2>Tags de paragrafos e quebra de linha</h2>
<p>A tag <strong>&lt;p&gt;</strong> Cria um paragrafo. Note que o navegador adiciona um estpaço(margem)
    antes e depois de cada parágrafo,</p>

    <p> este é um parágrafo. Ele ocupa um bloco de texto completo.</p>

    <p> Este é outro parágrafo

  <br> Esta é uma linha quebrada usando a tag <strong>&lt;br&gt;</strong>. O &lt;br&gt; è uma tag de linha simples, sem magem extra. </p>

<h2>Tags de Formatação e ênfase </h2>
<p>Essas tags servem para dar importãncia, ênfase ou outro tipo de Formatação ao texto.</p>

<p>
    <strong>&lt;b&gt;</strong> e <strong>&lt;strong&gt;</strong>: Texto em <b>negrito</b>
    e <strong>texto importante</strong>.<br>

    <strong>&lt;&gt;</strong> e <strong>&lt;em&gt;</strong>: Texto em <i>Itálico</i> e <em> texto com ênfase </em>.<br>

    <strong>&lt;u&gt;</strong>:<u> Texto sublinhado</u>.<br>

    <strong>&lt;small&gt;</strong>:<small>Texto menor</small>.<br>

    <strong>&lt;mark&at:</strong>: Texto <mark>marcado</mark>.<br>

    <strong>&it;sup&gt;</strong> e <strong>&it;sub&gt;</strong>: Texto com <sub>sobreescrito</sub> e com <sub>subscrito</sub> (bom para fórmulas:H<sub>2</sub>O).

    </sub><p>

        <h2>Tags de Citação e Código</h2>

        <p>
            <strong>&it;q&gt;</strong>: Uma citação curta, como: <q>Este é um exemplo de citação curta</q><br>

            <strong>&it;code&gt;</strong>: Para trechos de código pequenos, como a variável 'let' nome = "Turma";

        <pre>

                //Note que este código

            }
